# Coding guidelines

This is a markdown-based plugin project — all "source code" is SKILL.md files containing step-by-step instructions that Claude Code follows when a skill is invoked.

For quality standards, design principles, and writing conventions, see [skill-writing-guidelines.md](skill-writing-guidelines.md).

For actual code files (e.g. `scripts/deep-mode-harness/`), follow the coding principles in [coding-guidelines.md template](../../skills/init/templates/docs/coding-guidelines.md).
